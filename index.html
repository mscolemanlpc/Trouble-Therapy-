🔧 Firebase Setup Required

This game needs Firebase to sync between two players. Steps:

  1. Go to console.firebase.google.com
  2. Open your project → Add Web App → copy firebaseConfig
  3. Build → Realtime Database → Create (test mode)
  4. Paste your config into this HTML file
